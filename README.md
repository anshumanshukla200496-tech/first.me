Google Services Gallery

A responsive gallery webpage showcasing various Google services such as Gmail, Chrome, Maps, YouTube, Android, and more.

Live Demo

https://googlegallery20042006.netlify.app/

Project Overview

This project was built using HTML and CSS to practice web page structure, styling, and layout design. It displays different Google services in a clean card-based gallery format with images and descriptions.

Technologies Used

- HTML5
- CSS3
- Git & GitHub
- Netlify

Features

- Card-based gallery layout
- Images and descriptions for Google services
- Clean and simple user interface
- Beginner-friendly design
- Deployed online using Netlify

What I Learned

- Creating layouts using HTML and CSS
- Styling elements and managing spacing
- Working with images and content cards
- Using Git for version control
- Uploading projects to GitHub
- Deploying websites with Netlify

Future Improvements

- Add hover animations
- Improve typography and spacing
- Fix text and spelling errors

- Enhance overall UI/UX
